# Projeto de PLN - Sistema de Conversação para a Pandemia do COVID-19

## Descrição
Criação de um protótipo simples de chatbot como um sistema de conversação
para auxiliar as pessoas durante esse período de crise vivído na pandemia.

Feito para a disciplina de Processamento de Linguagem Natural (PLN) na UFABC.

Para mais informações, acesse o arquivo 'report.pdf' dentro da pasta 'docs'.

## Preparação
É necessário além de ter o Python instalado, também é preciso
instalar os pacotes necessários através do seguinte comando:
`pip install -r requirements.txt`.

E antes de rodar o programa, execute também o comando `python src/config.py`
para baixar e instalar recursos adicionais necessários do pacote NTLK.

## Execução
Para rodar o programa, execute o comando `python src/main.py` onde
será possível conversar com o _chatbot_ personalizado.
